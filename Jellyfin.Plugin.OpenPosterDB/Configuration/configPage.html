OpenPosterDB

Fetches posters, backdrops, logos and episode stills (with rating badges) from a self-hosted OpenPosterDB instance, by each item's IMDb / TMDB / TVDB id.

Your OpenPosterDB instance, e.g. https://openposterdb.com or http://localhost:3000 (no trailing slash needed).
Your 64-character OpenPosterDB API key.
Appended verbatim to every image request, e.g. lang=de&imageSize=large&badge_size=l. See the OpenPosterDB docs for the full parameter list.
